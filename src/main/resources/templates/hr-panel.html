HR Панель

Рекомендации HR-специалисту:

1. Создайте вакансию через кнопку «Создать вакансию».
2. Для каждой вакансии добавьте вопросы для тестирования.
3. После прохождения тестов кандидатами используйте раздел «Смотреть результаты кандидатов».
4. Нажмите на логин кандидата для просмотра его профиля и контактной информации.
5. Система автоматически оценивает кандидатов по результатам тестирования.
